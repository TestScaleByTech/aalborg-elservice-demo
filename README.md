# Aalborg El-service Demo

Fast 2-page Next.js demo for Aalborg El-service ApS.

## Stack

- Next.js App Router
- Tailwind CSS
- shadcn-style UI primitives
- Lucide icons

## Routes

- `/`
- `/kontakt`

## Local usage

```bash
npm install
npm run build
```

The app is configured for static export via `next build`, which writes deployable output to `out/`.
